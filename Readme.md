## First step, the .env

Please copy the `.env.example` file to `.env` and fill the variables with your own values.

```
cp .env.example .env
```

## To install and run the app

```
python -m venv venv
source ./venv/Scripts/activate
pip install pymongo bcrypt python-decouple
python ./gui.py
deactivate
```
